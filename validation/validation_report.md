# Relatório de Validação com Público-Alvo

## 1. Metodologia
A validação foi realizada no dia 01/12/2025, apresentando o MVP (Mínimo Produto Viável) instalado no smartphone via Expo Go para a usuária definida no perfil de público-alvo.

## 2. Testes Realizados
A usuária realizou as seguintes ações:
1.  Cadastro de nova conta (Login).
2.  Navegação pela lista de itens existentes.
3.  Publicação de um novo item para doação ("Livros Escolares").

## 3. Feedback Coletado (Qualitativo)
* **Facilidade de Uso:** A usuária considerou a interface limpa e as letras em tamanho adequado.
* **Cadastro de Item:** O processo foi considerado rápido, mas ela sentiu falta de poder tirar uma foto do produto diretamente no momento do cadastro (funcionalidade anotada para a próxima versão).
* **Confiança:** O fato de ver o e-mail do doador trouxe segurança sobre quem está doando.

## 4. Ajustes Implementados Pós-Validação
* Simplificação dos campos de cadastro para exigir apenas E-mail e Senha neste primeiro momento, reduzindo a fricção relatada no primeiro teste de acesso.

## 5. Conclusão
O aplicativo cumpre o propósito de conectar doador e donatário de forma simples. A validação confirmou que a barreira tecnológica é baixa e o produto é viável para o público do bairro.
